# \# 3D Models

# 

# Здесь лежат .glb модели для сайта.

# 

# Правила:

# \- Только .glb

# \- Только латиница

# \- Без пробелов и скобок

# \- Использовать kebab-case

# 

# Хорошо:

# \- dress.glb

# \- jeans-bottom.glb

# 

# Плохо:

# \- платье.glb

# \- dress FINAL.glb

# \- jeans(низ).glb

# 

# Перед добавлением:

# 1\. Оптимизировать модель

# 2\. Проверить открытие в viewer

# 3\. Проверить размер файла

# 

# Желательный размер:

# \- 0.5–8 MB
