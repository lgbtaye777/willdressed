# src/styles

CSS source.

Main file:

* `css/style.css` 

Use this folder when changing:

* layout
* typography
* buttons
* modals
* drawers
* catalog styling
* Fit Finder styling
* cart/checkout UI
* responsive behavior
* custom cursor visual states

Rules:

* Prefer scoped selectors.
* Avoid global `button`, `input`, `section`, `a` overrides unless intentional.
* Keep visual style dark, editorial, soft, and consistent.
* Do not introduce harsh white panels or browser-default controls.
* Reuse existing variables and button classes where possible.
* Check desktop and mobile.
* Respect low-power/reduced-motion classes.

Avoid adding unrelated visual refactors to functional fixes.
